# Python基于机器学习的文本情感系统 python1037

## 项目概述

本项目是一个基于Python和机器学习的文本情感分析系统。通过该系统，用户可以输入文本信息，系统将对其情感倾向进行分类。

## 技术栈

- 开发语言：Python 3.6.8
- 数据库：MySQL 5.7
- 数据库工具：Navicat 11
- 开发环境：PyCharm

## 功能模块

### 文本分类模块

- **功能介绍**：用户通过文本对话框输入信息，系统对输入的文本进行情感检测和运算。
- **操作流程**：用户在文本框中输入信息，点击“分类”按钮，系统对文本内容进行分析，并输出情感分析结果。

### 文本管理模块

- **功能介绍**：记录文本分类模块中的内容，并以列表形式展示。
- **展示内容**：显示用户输入的文本内容、机器判断的情感结果等。

## 系统角色

- 用户：可输入文本，查看情感分析结果。

## 运行环境

- 操作系统：兼容Windows、Linux等常见操作系统。
- Python环境：需预先安装Python 3.6.8版本。

## 部署步骤

1. 安装Python 3.6.8环境。
2. 安装MySQL 5.7数据库，并使用Navicat 11建立数据库连接。
3. 使用PyCharm或其他支持Python的IDE打开项目代码。
4. 配置相关环境变量和数据库连接信息。
5. 运行项目。
